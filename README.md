## üöÄ What is MOI?

MOI is a next-generation blockchain that solves the State Bloat problem through a **Participant-Centric** architecture. Unlike traditional blockchains, MOI uses an **Interaction State Machine (ISM)** that enables parallel execution and horizontal scaling.

### Key Features

- **Native Assets**: Protocol-level token support (no smart contracts needed!)
- **Contextual Execution**: Parallel processing without global bottlenecks
- **Coco Language**: Smart contract language for custom logic when needed
- **TDU (Transaction Data Unit)**: Efficient account state queries
